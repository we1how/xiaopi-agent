```
P(策略有效 | 近期表现) = P(近期表现 | 策略有效) × P(策略有效) / P(近期表现)

后验 = 似然 × 先验 / 证据
```

```python
import numpy as np

def bayesian_update(prior_prob, likelihood, evidence):
    """
    prior_prob: 先验概率 P(A)
    likelihood: 似然 P(B|A) 
    evidence: 证据 P(B)
    """
    posterior = (likelihood * prior_prob) / evidence
    return posterior

# 例子: 因子胜率更新
prior_win_rate = 0.55  # 历史胜率55%
# 假设: 如果因子真的有效，观察到3连胜的概率是80%
likelihood = 0.80
# 证据: 任意情况下观察到3连胜的概率（需要计算）
evidence = 0.55 * 0.80 + 0.45 * 0.20  # 全概率公式

posterior = bayesian_update(prior_win_rate, likelihood, evidence)
print(f"更新后胜率: {posterior:.2%}")
```

#### 🎯 学习方法

**核心概念理解 (30分钟)**

为什么用Beta分布建模胜率？
```
1. Beta分布定义在[0,1]区间 → 适合表示概率/胜率
2. Beta(α, β)的期望 = α/(α+β)
3. α可以看作"先验胜场数"，β看作"先验败场数"

例子:
- Beta(1,1) = Uniform(0,1) 完全不确定
- Beta(5,5) 期望50%，较保守
- Beta(10,2) 期望83%，较乐观
```

**手工计算练习 (15分钟)**
- 计算Beta(5,5)的期望、方差
- 假设观察到3胜2负，后验是什么？(答案: Beta(8,7))

**理解共轭先验 (15分钟)**
```
共轭先验的直观解释:
- 先验: Beta(α, β)
- 观测: N次试验中赢了k次
- 后验: Beta(α+k, β+(N-k))

这就像"先验是虚拟的历史数据"，观测到真实数据后直接相加
```

```python
import matplotlib.pyplot as plt
from scipy.stats import beta

# 定义三种先验
priors = [
    (1, 1, "弱先验: Beta(1,1)"),
    (5, 5, "中等: Beta(5,5)"),
    (10, 10, "强先验: Beta(10,10)")
]

# 假设观测到: 7胜3负
observed_wins, observed_losses = 7, 3

fig, axes = plt.subplots(1, 3, figsize=(15, 4))

for idx, (alpha, beta_param, title) in enumerate(priors):
    # 先验分布
    x = np.linspace(0, 1, 100)
    prior_pdf = beta.pdf(x, alpha, beta_param)
    
    # 后验分布
    posterior_alpha = alpha + observed_wins
    posterior_beta = beta_param + observed_losses
    posterior_pdf = beta.pdf(x, posterior_alpha, posterior_beta)
    
    # 绘图
    axes[idx].plot(x, prior_pdf, 'b--', label=f'Prior: Beta({alpha},{beta_param})')
    axes[idx].plot(x, posterior_pdf, 'r-', label=f'Posterior: Beta({posterior_alpha},{posterior_beta})')
    axes[idx].set_title(title)
    axes[idx].legend()
    axes[idx].set_xlabel('Win Rate')
    axes[idx].set_ylabel('Density')

plt.tight_layout()
plt.savefig('prior_comparison.png', dpi=150)
plt.show()

# 打印后验期望
for alpha, beta_param, title in priors:
    posterior_alpha = alpha + observed_wins
    posterior_beta = beta_param + observed_losses
    posterior_mean = posterior_alpha / (posterior_alpha + posterior_beta)
    print(f"{title}: 后验期望胜率 = {posterior_mean:.2%}")
```

```python
"""
贝叶斯因子更新模块
用于动态更新因子权重/置信度
"""
import numpy as np
from typing import Tuple, List
from scipy.stats import beta

class BayesianFactorUpdater:
    """
    使用Beta-Binomial共轭先验更新因子胜率估计
    """
    
    def __init__(self, prior_alpha: float = 5.0, prior_beta: float = 5.0):
        """
        初始化先验
        
        Args:
            prior_alpha: 先验"胜场数"，默认5表示中性
            prior_beta: 先验"败场数"，默认5表示中性
        """
        self.prior_alpha = prior_alpha
        self.prior_beta = prior_beta
        self.reset()
    
    def reset(self):
        """重置到先验状态"""
        self.alpha = self.prior_alpha
        self.beta = self.prior_beta
        self.history = []  # 存储历史观测
    
    def update(self, wins: int, losses: int) -> Tuple[float, float, float]:
        """
        根据观测数据更新后验
        
        Args:
            wins: 近期胜场数
            losses: 近期败场数
            
        Returns:
            (后验期望胜率, 后验alpha, 后验beta)
        """
        # TODO: 实现更新逻辑
        # self.alpha = ...
        # self.beta = ...
        
        posterior_mean = self.alpha / (self.alpha + self.beta)
        return posterior_mean, self.alpha, self.beta
    
    def get_confidence_interval(self, confidence: float = 0.95) -> Tuple[float, float]:
        """
        获取胜率的置信区间
        
        Args:
            confidence: 置信水平，默认95%
            
        Returns:
            (下限, 上限)
        """
        # 使用Beta分布的分位数
        lower = beta.ppf((1 - confidence) / 2, self.alpha, self.beta)
        upper = beta.ppf(1 - (1 - confidence) / 2, self.alpha, self.beta)
        return lower, upper
    
    def should_trade(self, threshold: float = 0.55) -> bool:
        """
        基于后验胜率判断是否启用该因子
        
        Args:
            threshold: 胜率阈值，默认55%
            
        Returns:
            True if 后验胜率 > threshold
        """
        current_win_rate = self.alpha / (self.alpha + self.beta)
        return current_win_rate > threshold


# 便捷函数：从IC序列批量更新
def update_from_ic_series(
    ic_series: np.ndarray,
    window: int = 20,
    prior_alpha: float = 5.0,
    prior_beta: float = 5.0,
    win_threshold: float = 0.0
) -> List[Tuple[float, float, float]]:
    """
    从IC序列计算滚动贝叶斯胜率
    
    Args:
        ic_series: 因子IC时间序列
        window: 滚动窗口长度
        prior_alpha, prior_beta: 先验参数
        win_threshold: 判定为"胜"的IC阈值
        
    Returns:
        每个时点的(后验胜率, alpha, beta)列表
    """
    updater = BayesianFactorUpdater(prior_alpha, prior_beta)
    results = []
    
    for i in range(len(ic_series)):
        if i < window:
            # 数据不足，使用先验
            results.append((
                prior_alpha / (prior_alpha + prior_beta),
                prior_alpha,
                prior_beta
            ))
        else:
            # 计算窗口内的胜负
            window_ic = ic_series[i-window:i]
            wins = np.sum(window_ic > win_threshold)
            losses = window - wins
            
            # 更新（注意：这里每次都reset，实际应该用累积更新）
            updater.reset()
            posterior_mean, alpha, beta_param = updater.update(wins, losses)
            results.append((posterior_mean, alpha, beta_param))
    
    return results
```

#### 🎯 学习方法

**核心概念 (30分钟)**

```
MLE核心思想:
给定观测数据 D = {x1, x2, ..., xn}，找到参数 θ 使得 P(D|θ) 最大

通俗解释:
"什么样的参数，最可能产生我观察到的数据？"

例子:
观测到10次股票涨跌: [涨, 跌, 涨, 涨, 跌, 涨, 涨, 涨, 跌, 涨] (7涨3跌)
MLE估计: 上涨概率 = 7/10 = 70%
```

**对数似然 (15分钟)**
```
为什么用对数似然？
- 原始似然: L(θ) = P(x1|θ) × P(x2|θ) × ... × P(xn|θ)
- 对数似然: log L(θ) = log P(x1|θ) + log P(x2|θ) + ... + log P(xn|θ)
- 好处: 乘法变加法，数值稳定，求导简单
```

- 答案: μ̂ = 样本均值, σ̂² = 样本方差（注意分母是n不是n-1）

```python
np.random.seed(42)
returns = np.random.standard_t(df=4, size=1000) * 0.015  # t分布，自由度4

# 2. 正态分布的MLE (10分钟)
def neg_log_likelihood_normal(params, data):
    """负对数似然: 正态分布"""
    mu, sigma = params
    if sigma <= 0:
        return 1e10
    # scipy.stats.norm.logpdf 返回对数概率密度
    log_likelihood = np.sum(stats.norm.logpdf(data, loc=mu, scale=sigma))
    return -log_likelihood

# 初始猜测
initial_guess = [np.mean(returns), np.std(returns)]

# 优化
result_normal = optimize.minimize(
    neg_log_likelihood_normal,
    initial_guess,
    args=(returns,),
    method='Nelder-Mead'
)

mu_mle, sigma_mle = result_normal.x
print(f"正态分布MLE: μ={mu_mle:.6f}, σ={sigma_mle:.6f}")

# 3. t分布的MLE (15分钟)
def neg_log_likelihood_t(params, data):
    """负对数似然: t分布"""
    df, loc, scale = params
    if df <= 2 or scale <= 0:
        return 1e10  # 无效参数返回大值
    log_likelihood = np.sum(stats.t.logpdf(data, df=df, loc=loc, scale=scale))
    return -log_likelihood

# 初始猜测
df_init = 5  # 自由度初始值
initial_guess_t = [df_init, np.mean(returns), np.std(returns)]

# 优化（带边界）
result_t = optimize.minimize(
    neg_log_likelihood_t,
    initial_guess_t,
    args=(returns,),
    method='L-BFGS-B',
    bounds=[(2.1, 100), (None, None), (0.001, None)]  # df>2, scale>0
)

df_mle, loc_mle, scale_mle = result_t.x
print(f"t分布MLE: df={df_mle:.2f}, loc={loc_mle:.6f}, scale={scale_mle:.6f}")

# 4. 模型对比: AIC和BIC (10分钟)
def calculate_aic_bic(log_likelihood, n_params, n_samples):
    """计算AIC和BIC"""
    aic = -2 * log_likelihood + 2 * n_params
    bic = -2 * log_likelihood + n_params * np.log(n_samples)
    return aic, bic

# 正态分布的log-likelihood
log_lik_normal = -result_normal.fun
aic_normal, bic_normal = calculate_aic_bic(log_lik_normal, 2, len(returns))

# t分布的log-likelihood
log_lik_t = -result_t.fun
aic_t, bic_t = calculate_aic_bic(log_lik_t, 3, len(returns))

print("\n模型对比:")
print(f"正态分布: AIC={aic_normal:.2f}, BIC={bic_normal:.2f}")
print(f"t分布:    AIC={aic_t:.2f}, BIC={bic_t:.2f}")
print(f"AIC差异: {aic_normal - aic_t:.2f} (负值表示t分布更好)")

# 5. 可视化对比 (10分钟)

plt.figure(figsize=(12, 5))

# 直方图

plt.hist(returns, bins=50, density=True, alpha=0.7, label='Data')
plt.plot(x, stats.norm.pdf(x, mu_mle, sigma_mle), 'r-', label='Normal MLE', linewidth=2)
plt.plot(x, stats.t.pdf(x, df_mle, loc_mle, scale_mle), 'g--', label='t MLE', linewidth=2)
plt.xlabel('Return')
plt.ylabel('Density')
plt.title('Return Distribution Fit')
plt.legend()

# Q-Q图检验尾部

stats.probplot(returns, dist=stats.t, sparams=(df_mle,), plot=plt)
plt.title('Q-Q Plot vs t-distribution')

plt.tight_layout()
plt.savefig('distribution_fit.png', dpi=150)
plt.show()

# 6. 尾部风险对比
normal_var95 = stats.norm.ppf(0.05, mu_mle, sigma_mle)
t_var95 = stats.t.ppf(0.05, df_mle, loc_mle, scale_mle)

print(f"\nVaR 95% 对比:")
print(f"正态分布: {normal_var95:.4f}")
print(f"t分布:    {t_var95:.4f}")
print(f"差异: {abs(t_var95) - abs(normal_var95):.4f} (t分布估计更保守)")
```

```python
@staticmethod
def estimate_prior_from_historical_rates(
    historical_win_rates: np.ndarray
) -> Tuple[float, float]:
    """
    用经验贝叶斯从历史胜率估计Beta先验参数
    
    Args:
        historical_win_rates: 历史胜率数组 (0到1之间)
        
    Returns:
        (alpha, beta) 先验参数
    """
    # 方法1: 矩估计 (简单快速)
    mean = np.mean(historical_win_rates)
    variance = np.var(historical_win_rates)
    
    if variance >= mean * (1 - mean):
        # 方差太大，返回中性先验
        return 5.0, 5.0
    
    # Beta分布的矩:
    # mean = α / (α + β)
    # variance = αβ / ((α+β)²(α+β+1))
    
    # 解方程得到α和β
    common_factor = mean * (1 - mean) / variance - 1
    alpha = mean * common_factor
    beta = (1 - mean) * common_factor
    
    return alpha, beta
```

```python
def get_posterior_stats(self) -> dict:
    """
    获取后验分布的统计信息
    
    Returns:
        {
            'mean': 期望胜率,
            'mode': 众数,
            'variance': 方差,
            'std': 标准差,
            'confidence_95': (下限, 上限)
        }
    """
    mean = self.alpha / (self.alpha + self.beta)
    
    # Beta分布的众数 (如果α>1且β>1)
    if self.alpha > 1 and self.beta > 1:
        mode = (self.alpha - 1) / (self.alpha + self.beta - 2)
    else:
        mode = None
    
    # 方差
    variance = (self.alpha * self.beta) / \
               ((self.alpha + self.beta)**2 * (self.alpha + self.beta + 1))
    
    # 置信区间
    lower, upper = self.get_confidence_interval(0.95)
    
    return {
        'mean': mean,
        'mode': mode,
        'variance': variance,
        'std': np.sqrt(variance),
        'confidence_95': (lower, upper),
        'alpha': self.alpha,
        'beta': self.beta
    }

def compare_to_prior(self) -> dict:
    """
    对比后验与先验的变化
    
    Returns:
        {
            'prior_mean': 先验期望,
            'posterior_mean': 后验期望,
            'shift': 变化量,
            'shrinkage': 收缩程度
        }
    """
    prior_mean = self.prior_alpha / (self.prior_alpha + self.prior_beta)
    posterior_mean = self.alpha / (self.alpha + self.beta)
    
    return {
        'prior_mean': prior_mean,
        'posterior_mean': posterior_mean,
        'shift': posterior_mean - prior_mean,
        'shrinkage': abs(posterior_mean - prior_mean) / abs(0.5 - prior_mean)
    }
```

```python
import matplotlib.pyplot as plt

def compare_methods(factor_ic, window=20, prior_alpha=5, prior_beta=5):
    """
    对比简单移动平均 vs 贝叶斯更新
    """
    n = len(factor_ic)
    
    # 方法1: 简单移动平均胜率
    sma_win_rate = factor_ic.rolling(window).apply(
        lambda x: (x > WIN_THRESHOLD).mean()
    )
    
    # 方法2: 贝叶斯更新胜率
    bayes_win_rate = []
    updater = BayesianFactorUpdater(prior_alpha, prior_beta)
    
    for i in range(n):
        if i < window:
            bayes_win_rate.append(prior_alpha / (prior_alpha + prior_beta))
        else:
            window_ic = factor_ic.iloc[i-window:i]
            wins = (window_ic > WIN_THRESHOLD).sum()
            losses = window - wins
            
            updater.reset()
            posterior_mean, _, _ = updater.update(wins, losses)
            bayes_win_rate.append(posterior_mean)
    
    bayes_win_rate = pd.Series(bayes_win_rate, index=factor_ic.index)
    
    return sma_win_rate, bayes_win_rate
```
