## 📰 项目 2: newsAI - 新闻智能分析系统

### 基本信息
- **GitHub**: https://github.com/we1how/newsAI
- **核心功能**: 财联社/微信/雪球新闻爬取 + AI分析对股票的影响
- **技术栈**: Python + OpenAI API (Moonshot/Volces) + akshare
- **输出**: Excel报告 + 邮件通知

### 核心组件

| 文件 | 功能 | 技术要点 |
|------|------|----------|
| `cls_news.py` | 财联社头条抓取 | 随机User-Agent、请求头伪装、签名生成 |
| `news_analyzer.py` | AI分析核心 | JSON修复、OpenAI API调用、影响评估 |
| `newsRss.py` | RSS订阅获取 | 多源新闻聚合 |
| `newsWechat.py` | 微信公众号监控 | wxauto自动化 |
| `xueqiu.py` | 雪球社区监控 | 投资者情绪分析 |
| `stock_analyzer.py` | 股票综合分析 | 多维度评估 |

### AI分析能力
- **输入**: 新闻标题 + 内容
- **输出**: 影响股票、影响程度、原因、总结
- **模型**: Moonshot/Volces API
- **容错**: JSON格式自动修复机制

### 项目关联性
- **与 lowValue 协同**: newsAI提供"催化剂"信息，lowValue提供超跌标的，形成完整交易闭环

## 📈 项目 3: lowValue - A股超跌反弹分析系统

### 基本信息
- **GitHub**: https://github.com/we1how/lowValue
- **核心功能**: 自动筛选大幅下跌后可能反弹的股票
- **技术栈**: Python + akshare + pandas + numpy
- **输出**: Excel分析报告

### 核心指标 (90天周期)

| 指标 | 说明 | 用途 |
|------|------|------|
| 大涨次数 (>6%) | 90天内涨幅超6%的次数 | 活跃度评估 |
| 价格最高点/日期/市值 | 90天内峰值 | 锚定参考 |
| 价格最低点/日期/市值 | 最高点后的最低点 | 入场参考 |
| 跌幅比 | (最低-最高)/最高 | 超跌程度 |
| 筹码获利比例 | 最低点时的获利盘 | 抛压评估 |
| 之后高点/涨幅比 | 最低点后的反弹高点 | 历史反弹验证 |

### 核心逻辑
```
大跌 → 缩量止跌 → 筹码集中 → 反弹信号
```

### 数据源
- **akshare**: 股票历史数据、筹码分布数据
- **反爬策略**: 随机User-Agent + headers + 随机延迟

### 项目关联性
- **与 newsAI 协同**: lowValue找到超跌标的，newsAI提供催化剂信息
- **与 QuantMunger 理念匹配**: 概率化、数据驱动的超跌策略

### 当前状态
- ✅ 核心功能已实现（数据采集+分析+Excel输出）
- ⚠️ 需要优化：回测验证、策略优化、可视化界面

### 团队协同建议
- **@QuantMunger**: 验证策略有效性，加入回测框架
- **@RigorousQA**: 检查代码健壮性、边界情况处理
