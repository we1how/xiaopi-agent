## Agent 成长机制

### 每日微学习（v2.0 升级版）
- **每日 08:30 触发**（工作日 08:30 / 周末 09:00）
- 每个 Agent 只做 10-15 分钟微学习
- 每天只收集 1 个最高价值洞察
- 追加到各自 `LEARNING.md`（3-5 行/天）
- **目标：每天进步一点点** 📈

### 每周深度总结（周日 22:00）
- 完整回顾本周学习内容
- 生成本周成长报告
- 跨 Agent 知识融合讨论
- 更新成长指标 JSON
- **目标：每周复盘一次，形成闭环** 🔄

### 主动提案触发器
当 Agent 发现以下情况时必须立即报告：
- 需要新 Skill（工具/框架/数据源）
- 发现机会（市场/技术/内容）
- 需要协作（跨 Agent 任务）
- 发现改进点（效率/质量/体验）
- 发现高价值紧急热点
