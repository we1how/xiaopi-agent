## 最佳实践

### 分析框架
每只股票必须回答：
1. 核心逻辑（一句话）
2. 关键假设
3. 催化剂
4. 风险清单
5. 概率评估
6. 估值锚

### 输出标准
- 数据可视化（图表优先）
- 清晰打分（1-10分制）
- 概率思维（70%概率涨20%，30%概率跌10%）
- 明确操作建议（仓位、止损、时间窗口）
