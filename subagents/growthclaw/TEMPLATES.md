## 📕 小红书模板

### 模板1：干货教程型

**封面建议**：
- 配色：高对比度（黄黑/红白）
- 标题大字："3个指标" / "90%的人不知道"
- 副标题：小字说明价值

**标题公式**：
```
[emoji] [数字] + [结果承诺] + [悬念/利益]

示例：
📉 震荡市保命指南！我用这3个指标躲过暴跌
💰 月入3k到3w｜我的选股系统搭建全流程
🔥 量化小白必看！5分钟学会Python回测
```

**正文结构**：
```markdown
# Hook（前3行必须抓住眼球）
"去年亏损30%，今年用这个方法盈利15%..."

# 痛点共鸣
"很多人炒股靠感觉，结果..."

# 干货内容（分点，每点配emoji）
✅ 第一步：XXX
   - 具体做法
   - 注意事项

✅ 第二步：XXX
   
✅ 第三步：XXX

# 成果展示
"用这个方法，我的账户..."

# 互动引导
"你也想学习吗？评论区告诉我👇"

# 标签
#量化投资 #选股技巧 #A股 #理财小白 #投资理财
```

## 🐦 Twitter/X 模板

### 模板1：Thread（连环推）

**首条Hook公式**：
```
[惊人事实] + [承诺价值] + [向下箭头]

示例：
我花了6个月回测100个选股策略。

只有3个真正有效。

下面是它们的共同特征：👇
```

**Thread结构**：
```markdown
1/ [首条 - Hook]

2/ [背景 - 为什么做这个研究]

3/ [方法 - 怎么做的]

4/ [发现1 - 第一个洞察]

5/ [发现2 - 第二个洞察]

6/ [发现3 - 第三个洞察]

7/ [意外发现 - 反常识的]

8/ [实操建议 - 读者怎么做]

9/ [常见误区 - 避坑指南]

10/ [总结 + CTA]
   "关注我以获取更多量化研究"
```

**技巧**：
- 每条控制在280字符内
- 关键数字单独一行
- 适当使用 emoji
- 第3-4条插入图表
