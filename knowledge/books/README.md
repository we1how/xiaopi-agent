## 使用指南

### 添加新书籍/论文

1. **确定分类**：根据主题放入对应文件夹
2. **创建文件**：使用模板创建 Markdown 文件
3. **更新索引**：在本文件中添加条目
4. **连接关联**：标注与其他书籍/项目的关联

### 文件命名规范

```
《书名》→ 书名.md（简化）
《An Opinionated Guide to ML Research》→ an-opinionated-guide-to-ml-research.md
《股票大作手回忆录》→ 股票大作手回忆录.md
```
