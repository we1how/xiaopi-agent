**📊 Dynamic-weight AMMs套利效率分析**
> 来源: Arxiv:2602.22069

**关键洞察**：
- TFMMs（动态权重AMM）通过预言机权重更新 + 套利者荷兰式拍卖机制，在L2上能达到或超越CEX再平衡效率
- LVR（Loss-vs-Rebalancing）基准可量化AMM再平衡成本，Base链上效率显著优于主网
- **应用价值**：在Stock Platform中引入LVR类指标评估策略调仓成本

- 支持用户自定义指标组合

**技术路径**：
```python
# 细粒度指标层
indicators = {
    "normalized_macd": calculate_normalized_macd(data),
    "bollinger_z": calculate_bollinger_z(data),
    "kdj_j": calculate_kdj(data)["j"]
}

# 策略组装层
strategy = combine_indicators(indicators, weights=[0.4, 0.3, 0.3])
```
